Financial Academia is a web-based financial literacy platform designed to make learning finance simple and engaging. The application provides different user experiences based on age, ensuring appropriate content and interactions for all users.

Users below 18 are guided through a gamified learning interface with quizzes, daily tasks, and reward-based activities to build basic financial awareness. Users 18 and above gain access to more advanced features such as stock market analysis games, financial task planning, and interactive challenges.

The platform includes secure OTP-based authentication, a personalized dashboard, progress tracking, and a reward system to encourage consistent learning. Financial Academia focuses on delivering an intuitive, age-appropriate, and interactive approach to financial education.

## Tech Stack
- **React Js**: Frontend Development Framework
- **Node Js, Express Js, Flask, FastApi**: Backend Development Framework
- **Web-Rtc TypeScript**: Video Streaming and chatting with live Q&A
- **yfinance**: Latest Stock shares information.
- **NLP and RAG for making daily Q&A**: Daily Q&A for financial knowledge.
- **Mistral 8x7B LLM**: Language model used for natural language processing tasks.
- **RAG with QDrant Vector DB**: Integrates RAG model and QDrant Vector DB for real-time query resolution.
  
## Installation Steps

### Frontend
1. Rename all `.env.sample` files to `.env` and fill in the required data.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
 
### Backend
1. Navigate to the server directory: `cd server`
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
